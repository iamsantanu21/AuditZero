# Digital Personal Data Protection Act, 2023

## What it is
India's comprehensive data protection law that governs the processing of digital personal data. It establishes rights of data principals (individuals) and obligations of data fiduciaries (organizations processing personal data).

## Who it applies to
Any entity processing digital personal data in India, or processing data of individuals in India regardless of where the entity is located.

## Key obligations
- Obtain free, specific, informed, and unambiguous consent before processing personal data
- Process data only for lawful purposes specified at the time of consent
- Appoint a Data Protection Officer (DPO) for Significant Data Fiduciaries
- Implement reasonable security safeguards to protect personal data
- Notify the Data Protection Board and affected individuals of data breaches
- Honour data principal rights: access, correction, erasure, grievance redressal, and nomination
- Restrict cross-border data transfers to government-notified countries only
- Delete personal data when purpose is fulfilled or consent is withdrawn
- Obtain verifiable parental consent for processing children's data
- Comply with Data Protection Board orders and audits

## Maximum penalty
Up to ₹250 crore per contravention

## Enforcement deadline
Sections notified on a rolling basis; Board constitution in progress (2024-2025)
