# Four Labour Codes, 2019-2020 (Compliance Handbook)

## What it is
Consolidated compliance handbook covering all four labour codes — Code on Wages (2019), Industrial Relations Code (2020), Occupational Safety, Health & Working Conditions Code (2020), and Code on Social Security (2020).

## Who it applies to
All employers and establishments in India (specific applicability thresholds vary by Code).

## Key obligations
- Pay wages above the prescribed minimum wage (floor wage set by Central Government)
- Maintain electronic registers and records as prescribed
- Provide appointment letters to all employees
- Constitute Internal Complaints Committee (ICC) under PoSH provisions
- Register establishment on the Shram Suvidha Portal
- Contribute to Employee Provident Fund (EPF), ESI, and Gratuity as applicable
- File unified annual return through the Shram Suvidha Portal
- Comply with occupational safety and working conditions standards
- Provide maternity benefits and leave as prescribed
- Follow prescribed procedures for retrenchment, lay-off, and closure

## Maximum penalty
Up to ₹3 lakh fine + imprisonment (varies by Code and violation)

## Enforcement deadline
Codes enacted 2019-2020; Central Rules notified — awaiting State rules for full implementation
