# IRDAI Information and Cyber Security Guidelines, 2023

## What it is
Comprehensive cybersecurity guidelines for insurance companies covering IT governance, risk management, data classification, access controls, incident response, cloud security, and third-party risk management across all insurance operations.

## Who it applies to
All insurance companies (life, general, health, reinsurers) and insurance intermediaries regulated by IRDAI.

## Key obligations
- Establish Board-approved Information and Cyber Security (ICS) Policy
- Appoint a Chief Information Security Officer (CISO)
- Implement data classification framework and data loss prevention controls
- Conduct annual VAPT by CERT-In empanelled auditors
- Report cyber incidents to IRDAI and CERT-In within 6 hours
- Implement multi-factor authentication for all critical systems and remote access
- Maintain Security Operations Centre (SOC) for continuous monitoring
- Conduct cyber security awareness training for all employees quarterly
- Implement cloud security controls as prescribed for cloud-hosted applications
- Conduct annual information security audit and submit report to IRDAI

## Maximum penalty
Up to ₹1 crore per violation + regulatory action

## Enforcement deadline
Effective from April 24, 2023
