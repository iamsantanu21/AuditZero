# RBI Cybersecurity Framework for Banks

## What it is
Framework requiring banks to implement a Board-approved Cyber Security Policy, establish a Security Operations Centre (SOC), conduct regular vulnerability assessments, and report cyber incidents to RBI within specified timelines.

## Who it applies to
All scheduled commercial banks, including foreign banks operating in India.

## Key obligations
- Establish a Board-approved Cyber Security Policy distinct from IT/IS policy
- Set up a Cyber Security Operations Centre (C-SOC)
- Conduct continuous surveillance and real-time monitoring of cyber threats
- Perform regular Vulnerability Assessment and Penetration Testing (VAPT)
- Report cyber incidents to RBI within 2-6 hours based on severity
- Implement multi-factor authentication for critical systems
- Ensure network segmentation and data leak prevention controls
- Conduct periodic cyber security audits by independent auditors
- Implement robust access management and privileged access controls
- Maintain an updated Cyber Crisis Management Plan (CCMP)

## Maximum penalty
Regulatory action including penalties up to ₹50 crore and supervisory directions

## Enforcement deadline
In force since June 2, 2016
